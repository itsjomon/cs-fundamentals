# CS Fundamentals  
> **🌟 Star this repository if you find it useful!**

This repository serves as a concise reference for essential Computer Science subjects, curated specifically for students, interview preparation, and revision. It includes high-quality PDF notes covering core theoretical topics that every CS student or aspiring developer should know.

## What's Inside?  

- **Operating Systems (OS)**  
  Notes include OS basics, types of operating systems, architecture, process management, process states, CPU scheduling algorithms, inter-process communication,deadlocks, memory management, virtual memory,
  and file systems.<br>
  [Read more about Operating Systems on GeeksforGeeks](https://www.geeksforgeeks.org/operating-systems/operating-systems/)  

- **Computer Networks (CN)**  
  Covers the OSI and TCP/IP models, switching techniques, transmission media, IP addressing, subnetting, routing algorithms, error detection methods, TCP vs UDP, DNS, HTTP/HTTPS,
  and fundamental network security concepts.<br>
  [Read more about Computer Networks on GeeksforGeeks](https://www.geeksforgeeks.org/computer-networks/basics-computer-networking/)  

- **Database Management Systems (DBMS)**  
  Includes database concepts, ER diagrams, relational model, relational algebra, normalization (1NF, 2NF, 3NF, BCNF), functional dependencies, indexing, hashing, transactions, concurrency control,
  and ACID properties.<br>
  [Read more about DBMS on GeeksforGeeks](https://www.geeksforgeeks.org/dbms/dbms/)  

- **SQL**  
  Notes cover SQL basics, DDL/DML/DCL commands, SELECT queries, WHERE clause, operators, aggregate functions, GROUP BY and HAVING, joins, subqueries, views, indexes, and transaction control statements.<br>
  [Read more about SQL on GeeksforGeeks](https://www.geeksforgeeks.org/sql-tutorial/)
